# famicomFC红白机2400游戏合集珍藏版

> 红白机全称是任天堂 Family Computer (简写成Famicom或红白机或FC)，任天堂红白机－FC（Family Computer）是日本任天堂公司1983年生产的游戏主机，现在很多游戏的前身就是来自于FC，FC为游戏产业做出了相当大的贡献，甚至可以说FC游戏机是日本游戏产业的起点，FC也曾在80、90年代风靡中国大陆，那个时候也有很多人管它叫红白机，相信很多玩家都有在童年时代玩红白机游戏的经历，也有很多玩家就是从这个神奇的主机开始了自己的游戏生涯。

![红白机](images/fc.png)

**FC红白机游戏全套3000个，包含100多个汉化版游戏和所有中文游戏、和115个精选的最经典游戏，所有游戏都单独存放，方便有选择性的下载，所有游戏rom都进行了精心整理，并使用中文名，方便你瞬间找到需要的游戏
内附模拟器(VirtuaNES 092绿色版,直接玩游戏)、模拟器使用教程、以及部分游戏密技、作弊码，收藏怀旧者必下！**

## 《超惑星战记》
·连续取得宠物法
在有坚梯的地方去得宝物后，先到外面去，再进入里面，如此，刚才得到的宝物又出现了，利用此法可骗得许多宝物。
·用手榴弹轰炸首领的办法
当第二关的首领出现时，把手榴弹投入它的要害——嘴中，然后按开始键使之暂停。此时画面不动，但可听到首领受伤后的惨叫，随
后按开始键解除暂停，手榴弹立即便会炸死首领。此法可用来对付第二、四、七关首领。

## 《大盗五佑卫门2》
·海面步行术
在第二关的阿波国，必须在海面上跳跃前进，如果跳跃失败而掉入海中，就会死亡。其实有一个既轻松又安全的方法：站在海岸的最
上面，然后按方向键右键，即可在海上行走，无论怎么都不会掉下去。
·复活术
两人玩时，如果其中一人被打死，另一人要赶快进入附近的商店，然后又赶紧从商店里走出来，这样被打死的伙伴又复活了。 ·地狱脱身术
发现地藏王菩萨，如果用烟斗敲打他的头部，你将会掉落到地狱中去。如果你想从地狱中解脱出来，就要走到右边的地方，那里有个
红鬼，敲打红鬼的头部，就可以从地狱里脱身了。
·走捷径法
游戏开始时，首先要走进地狱，出了地狱后，在高中出现的画面上跳跃，便会在地面上出现隐藏的楼梯入口。这就是通往村长住宅的
捷径。进入楼梯入口后，会有一条道路，走进去，就达到村长住宅了。

· 暗道破壁法
依上面的方法来到村长住宅后，先别急着去攻打村长，检查一下村长住宅的墙壁。走到最左边的墙壁前，按住方向键上，再按A、B，
就可以破壁进入一个暗道。这是一条直线暗道，可以一边捡钱，一边前进，这样可以获得500两银子。
